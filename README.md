# Techify

Techify is a modern, minimalist e-commerce platform for curated tech gear, inspired by Nordic lifestyle and aesthetics. 

## Features
- **User Dashboard**: Track orders, view account details, and manage wishlist.
- **Admin Panel**: Complete content management system for products, orders, users, and customer messages.
- **Product Catalog**: Dynamic filtering by Categories, Brands, and Price range.
- **Shopping Cart**: Real-time cart updates with a seamless checkout process.

## Tech Stack
- **Frontend**: Angular, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT-based authentication

## Getting Started

### Prerequisites
- Node.js (v18+)
- Angular CLI
- MongoDB (Local or Atlas)

### Installation
1. Install dependencies for both client and server:
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```

2. Start the development servers:
   ```bash
   # Terminal 1: Start Angular frontend
   cd client
   npm start

   # Terminal 2: Start Express backend
   cd server
   npm start
   ```

3. Open your browser and navigate to `http://localhost:4200`
